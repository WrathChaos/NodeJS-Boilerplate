# Installation & Run

It requires babel-cli to work with ES6 😇

```
npm install babel-cli -g --save
```

As always 😂
```
npm install
```

#### RUN  😍
```
npm start
```


## Docker Build & Run

```
sh build.sh
```

```
docker-compose -f docker-compose.yml -f docker-compose.yml up -d nodejs-boilerplate
```
